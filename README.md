TVShow

This project is a TV Shows & Movies web application built with **Next.js (App Router)**.  
It lets users browse **films, series, and actors**, view detailed pages for each item, manage a **Likes** list, and use **dark/light mode** with a responsive UI.

Features:
1- Browse Movies, Series, and Actors  
2- Dynamic detail pages for each item (e.g. `/films/[id]`, `/series/[id]`, `/person/[id]`)  
3- Likes page to save and manage favorites  
4- Sign In / Sign Up pages  
5- Dark and light theme toggle  
6- Responsive design for desktop and mobile

Technologies and Libraries:
1- JavaScript / React  
2- Next.js (App Router)  
3- Tailwind CSS  
4- Redux Toolkit + React Redux  
5- React Hook Form  
6- Radix UI / shadcn-ui  
7- next-themes

How to Run:
1- `npm install`  
2- `npm run dev`
